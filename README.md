# Action Agent Studio

一个基于 Electron 的 AI Agent 原型，用来录制网页里的用户动作，理解这些动作代表的任务目标，并在之后回放执行。

## 当前能力

- 录制网页里的点击、输入、滚动、快捷键和页面跳转
- 将动作压缩成更稳定的步骤序列
- 本地生成“动作理解”结果，包括目标、步骤、风险和自动化建议
- 可选接入 OpenAI 兼容接口，使用 LLM 增强理解质量
- 导出/导入 JSON 流程文件
- 对已录制流程执行基础回放

## 安装

```bash
npm install
```

## 运行

```bash
npm start
```

## 使用方式

1. 在左侧输入目标网址并打开页面。
2. 点击“开始录制”。
3. 在中间网页区域完成一段真实操作。
4. 点击“停止”。
5. 点击“理解动作”，获得 AI 对这段动作的解释。
6. 点击“回放流程”，让应用重复执行动作。

## 注意事项

- 当前版本重点覆盖“网页内动作录制与复现”，不是系统级全桌面录制器。
- 回放依赖 DOM 选择器，如果页面结构变化较大，动作可能失效。
- 密码输入在录制阶段会被遮蔽成 `********`，避免直接存入导出文件。
- 如果要做成生产级产品，建议继续加入元素等待、失败重试、视觉定位、权限系统和任务编排。
